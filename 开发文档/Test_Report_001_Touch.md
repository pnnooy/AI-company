# 测试报告 #001 — TTP223 触摸传感器 + RGB LED

**日期**: 2026-05-21  
**硬件**: TTP223 模块 ×2, PC4(左) / PC5(右), 3.3V / GND  
**测试环境**: HSI 64MHz, 串口 115200-8-N-1

---

## 测试结果

| 测试项 | 结果 | 说明 |
|--------|------|------|
| 左触摸 TAP (单击) | ✅ 通过 | 响应灵敏，串口输出正常 |
| 右触摸 TAP (单击) | ✅ 通过 | 响应灵敏 |
| 左触摸 HOLD (长按>1s) | ⚠️ 不舒适 | 可触发但手感偏长，需优化阈值 |
| 右触摸 HOLD (长按>1s) | ⚠️ 不舒适 | 同上 |
| DOUBLE (双触) | ❌ 几乎无法触发 | 需两人手同时触，实际使用不可行 |
| 系统状态切换 | ✅ 通过 | IDLE→ACTIVE→INTERACT 链路正确 |
| 30s 超时回退 | ✅ 通过 | INTERACT→ACTIVE 正常 |
| 串口文本输出 | ✅ 通过 | 人类可读，调试友好 |

## 待优化项

1. **HOLD 阈值**: 当前 1000ms，建议降至 600-800ms
2. **DOUBLE 双触**: 当前要求两个 PC4+PC5 同时 HIGH。改为：500ms 内先后触发两个传感器即算双触
3. **消抖时间**: 当前 50ms，表现正常可保留
