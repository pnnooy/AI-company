# 角色 D：表情素材 AI 生成

> **负责人**: D
> **工期**: 预计 1 天
> **前置条件**: 可访问 AI 图像生成工具（如 GPT-4o / Gemini / Midjourney / 其他生图模型）

---

## 一、任务总览

使用 AI 图像生成工具，为机器人设计 8 种表情素材。将现有的参考图片投喂给 AI，让 AI 生成风格统一的表情图片。

**这是纯独立任务**，不依赖任何人。你只需要和 AI 对话、筛选图片、整理文件。

---

## 二、你要做什么

```
现有表情参考图（faceset1/ 中的图片或其他参考图）
       ↓ 投喂给 AI 生图工具
AI 生成 8 种表情 × 3 张候选 = 24 张图片
       ↓ 你筛选每类表情最佳的一张
最终交付 8 张表情图片
       ↓ 存入项目文件夹
交给 A/B 后续处理（裁剪、转换格式、写入固件等由其他人做）
```

## 四、图片要求（很简单）

| 要求 | 说明 |
|------|------|
| 人物居中 | 角色在画面正中，不要偏左偏右 |
| 大小一致 | 同一批图片中角色大小尽量统一（不用很精确，看着差不多就行） |
| 比例 1:1 | 正方形即可，分辨率不限制（AI 生成啥就收啥） |
| 格式 | JPG 或 PNG 都可以，**不需要你做任何后期处理** |
| 风格统一 | 所有表情看起来是同一个角色（在 prompt 里强调这一点） |
| 背景 | 最好是纯色或简单背景，深色/黑色更优（但不强制） |

**明确不需要你做的**：
- ❌ 不需要管 80×80 分辨率
- ❌ 不需要管 RGB565 颜色格式
- ❌ 不需要跑 png2rgb565.py
- ❌ 不需要转 C 数组
- ❌ 不需要在板子上烧录验证

这些技术处理由 A 或 B 后续完成。你只负责用 AI 生成高质量的表情图片。

## 五、AI 生图工作流程

### Step 1: 准备参考素材

项目中已有的参考图片在 `firmware/Assets/faceset1/` 目录下，挑几张风格你喜欢的投喂给 AI。

也可以自己在网上找参考图（卡通风格、Line Friends 风格、像素风等），让 AI 参考风格。

### Step 2: 写 Prompt

核心 Prompt 要点（根据你用的 AI 工具调整）：

```
你是一个角色设计师。请为一个桌面机器人设计表情素材。

角色设定：可爱的卡通小机器人/小怪物，圆润简约风格。

风格参考：[把参考图贴在这里]

要求：
- 正方形构图，1:1 比例
- 角色在画面正中
- 纯色或简单深色背景
- 所有图片保持同一角色、同一画风、大小一致

请生成以下表情：[某种表情的描述]
```

### Step 3: 逐类生成

按顺序生成（每类 3 张）：

1. **普通 NORMAL**："中性表情，平静自然地看着前方，圆眼睛，小嘴巴呈直线或微弯"
2. **开心 HAPPY**："开心愉悦的表情，笑眼弯曲像 ^_^，嘴巴上弯大笑"
3. **专注 FOCUS**："认真专注的表情，眼睛微微眯起，嘴巴紧闭，像在思考"
4. **生气 ANGRY**："生气的表情，倒八字眉，眼神锐利，嘴巴下弯"
5. **休眠 SLEEP**："睡觉的表情，双眼闭合呈横线，嘴巴小圆圈像打呼"
6. **惊讶 SURPRISE**："惊讶的表情，双眼瞪大，嘴巴张成 O 型"
7. **难过 SAD**："难过的表情，八字眉，眼睛下垂，嘴巴下弯，可加泪滴"
8. **爱心 LOVE**："心动的表情，爱心形状的眼睛，微笑嘴巴，脸颊有腮红"

### Step 4: 筛选 + 重命名

1. 每类 3 张中，挑出最好的一张标记为"首选"
2. 所有图片统一放到一个文件夹
3. 文件名清晰可辨（如 `emo_happy_v1.png`，或 `表情_开心_v1.png`）

### Step 5: 交付

把 24 张图片打包发给团队，附一个简短说明：
- 每类表情哪张是首推
- 用的什么 AI 工具 + 什么 prompt（方便后续复现或调整）

## 六、风格选择建议

### 方向 1：可爱卡通

```
"cute cartoon robot character, round face, big expressive eyes,
simple clean design, like Line Friends or Kakao Friends,
solid dark background, square composition, consistent style"
```

### 方向 2：像素风

```
"pixel art character, 8-bit style, cute robot face,
black background, square canvas, game sprite style,
consistent design across all expressions"
```

### 方向 3：极简图标风

```
"minimalist flat design character icon, simple shapes,
bold colors, clean lines, dark background,
app icon style, consistent across all variants"
```

**建议**：三种方向各生成一组 NORMAL 表情作为试水，团队投票选出风格方向后，其余 7 种表情统一用该方向生成。这样避免了 24 张全部生成后风格被否定的风险。

## 七、AI 工具推荐

| 工具 | 优势 | 限制 |
|------|------|------|
| **GPT-4o** (ChatGPT) | 图像生成质量好，理解中文，可直接对话调整 | 需要 Plus 订阅 |
| **Gemini** (Google) | 免费额度大，图像生成质量不错 | 需要 Google 账号 |
| **Midjourney** | 画质最好 | 需要 Discord，英文 prompt，付费 |
| **通义万相** (阿里) | 免费，中文支持好 | 画风偏国风 / 写实，卡通效果待测 |
| **文心一格** (百度) | 免费额度 | 画质一般 |

**推荐先用 GPT-4o 或 Gemini** — 可以直接对话式调整，不满意就让它重新生成，效率最高。

## 八、常见问题

| Q | A |
|---|---|
| AI 生成的风格不一致怎么办？ | 给 AI 看上一张满意的图作为参考，让它"保持同一角色、同一画风" |
| 生成的图不是 1:1 怎么办？ | 在 prompt 里强调 "square 1:1 aspect ratio"，多试几次 |
| 角色偏左偏右不居中？ | "character centered in frame" 加上去 |
| 背景不是纯色？ | 只要不是太花哨就行，后续 A/B 会处理 |
| 每类需要严格 3 张吗？ | 不需要死板，如果第一张就很满意，2 张也行；如果都不满意，5 张也行 |
| 需要我在板子上测试吗？ | 不需要，你只负责生成图片文件 |
