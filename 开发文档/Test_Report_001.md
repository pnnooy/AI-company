# 测试报告 #001 — 外设模块验证

**日期**: 2026-05-21  
**MCU**: STM32F103VET6, HSI 64MHz, Keil MDK V5  
**烧录器**: Fire-Debugger (野火 DAP)  
**串口工具**: SSCOM, COM6, 115200-8-N-1

---

## 1. 串口通信 (USART1)

| 测试项 | 结果 | 说明 |
|--------|------|------|
| TX 发送 | ✅ | printf 重定向正常, 输出启动信息 |
| RX 接收 | ✅ | 文本命令解析正常, 需用 SSCOM (关 DTR/RTS) |
| 时钟 | ⚠️ | 当前 HSI 64MHz. HSE 晶振未起振, 待排查 |
| 工具兼容 | ⚠️ | 野火调试助手/部分串口工具会触发 CH340 自动下载电路导致 MCU 复位 |

**关键发现**: 野火板载 CH340, USB 转串口插口已内部连接 PA9/PA10, 无需外接模块. DTR/RTS 会引起自动下载, 必须用 SSCOM 等无流控工具.

## 2. RGB LED (板载全彩LED)

| 测试项 | 结果 | 说明 |
|--------|------|------|
| 基本颜色控制 | ✅ | 红/绿/蓝/橙/白 纯色准确 |
| 呼吸效果 | ✅ | 三角波 PWM, 流畅 |
| 串口命令 `led R G B` | ✅ | 实时切换, 无闪烁 |
| 启动验证 | ✅ | 3 闪确认初始化完毕 |

**硬件参数**: PB5(R-TIM3_CH2), PB0(G-TIM3_CH3), PB1(B-TIM3_CH4). 共阳LED, 需 TIM_OCPOLARITY_LOW. TIM3 需 Partial Remap (PB5). Prescaler=1999, Period=255.

| 30s 超时回退 | ✅ | INTERACT→ACTIVE 正常 |

## 4. MPU6050 姿态传感器

| 测试项 | 结果 | 说明 |
|--------|------|------|
| 通信链路 | ✅ | 软件 I2C (PA11=SCL, PA12=SDA) |
| SHAKE 检测 | ✅ | 晃动/旋转, gyro > 200°/s |
| FALL 检测 | ✅ | 快速突然移动, accel < 0.4g 或 > 2.5g |
| 串口命令 `mpu` | ✅ | 输出 accel/gyro/pose |
| `mpuoff`/`mpuon` | ✅ | 停止/恢复姿态轮询, 拔模块后关掉 |
| 模块上拉 | ✅ | MPU6050 模块自带 4.7kΩ |

**关键发现**: FSMC NADV 与 I2C1_SDA 共用 PB7, 硬件冲突. **不可使用硬件 I2C1**. 野火 PDF 第47章推荐软件 I2C 方案.

### MPU6050 检测阈值

| 姿态 | 条件 | 值 |
|------|------|------|
| SHAKE | 角速度 | > 200°/s |
| FALL | 加速度偏离1g | < 0.4g 或 > 2.5g |
| STABLE | 正常 | 0.4~2.5g |
