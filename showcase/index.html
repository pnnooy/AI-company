About

项目概述

一款面向大学生的桌面级学术助手机器人，通过多模态交互实现陪伴式学习体验

认识皮皮

皮皮是一款桌面学术助手机器人，基于 STM32F103VET6 微控制器打造。它拥有 8 种动态表情（通过 ILI9341 LCD 呈现）、RGB 氛围灯光、电容触摸感知、六轴姿态检测（MPU6050）以及 NFC 卡片识别（MFRC-522）。

通过 USART 与 PC 端 AI 引擎通信，皮皮能够理解用户的语言输入，做出智能回复，并配合表情和灯光营造生动的交互体验。支持摄像头人脸情绪识别，实现双向情感交互闭环。

Architecture

系统架构

STM32 固件 → UART 通信 → PC AI 引擎 → 前端交互

STM32 固件层

HAL 库 · 非阻塞 FSM · ILI9341 LCD 驱动 · MPU6050 · MFRC-522 · TTP223 触摸 · RGB PWM

UART 通信层

USART1 · 115200bps · 二进制协议 + CRC 校验 · PC ↔ STM32 双向通信

PC AI 引擎层

Python · LLM 对话 · 状态机决策 · 情绪识别 (ONNX) · 摄像头 CV · 前端 Web UI

交互呈现层
